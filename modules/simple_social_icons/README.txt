INTRODUCTION
------------

This Module provides simple configurable social icons as blocks.
Icons can be configurable with size, radius, colors and padding.
for more info visit https://www.drupal.org/project/simple_social_icons

REQUIREMENTS
------------

This module does not have any dependency.

INSTALLATION
------------

* Install as you would normally install a contributed Drupal module. See:
  https://www.drupal.org/documentation/install/modules-themes/modules-8
  for further information.

CONFIGURATION
-------------

* Copy the "simple_social_icons" directory under "modules/";
* Go to "admin/modules" and enable "simple_social_icons" module;
* Go to "admin/structure/block/" select one of the region and place the
  block under that and configure the settings.
